# 真实键盘逐字输入器

一个 Windows 桌面小工具：在窗口里粘贴文字或代码，点击“运行”后，再点击任意目标输入框，程序会通过 Windows `SendInput` 自动逐字输入内容。

## 功能

- 中文界面
- 支持中文、英文、符号、换行、空格、缩进和代码格式
- 可选择输入格式：自动识别、原样输入、代码模式
- 可选择按键方式：Unicode 逐字、智能模式、英文物理按键、兼容粘贴
- 支持设置每字输入间隔和点击目标后的延迟
- 支持运行时最小化窗口
- 按 `Esc` 可取消

## 使用方法

1. 打开 `dist/RealKeyboardTyper.exe`。
2. 粘贴要输入的文字或代码。
3. 保持默认“Unicode逐字（中文推荐）”。
4. 点击“运行”。
5. 点击目标软件里的输入框，等待自动输入完成。

如果某个软件不接收中文输入，可以把“按键方式”切换为“兼容粘贴（备用）”。

## 重新编译

本项目使用 Windows 自带的 .NET Framework C# 编译器即可编译，不需要安装 .NET SDK。

```powershell
.\build.ps1
```

生成文件会输出到：

```text
dist\RealKeyboardTyper.exe
```

## 注意

“真实外接键盘”本身只发送按键码，中文通常需要输入法把拼音、五笔等组合成汉字。本工具为了稳定输入中文，默认使用 Windows `SendInput` 的 Unicode 按键事件；如果目标软件屏蔽这种输入，可以使用“兼容粘贴（备用）”模式。
